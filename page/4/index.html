C++基础

基本概念

  • 命名空间
    • using 编译指令 using namespace std; 引用std的全部实体
    • 域限定符 namespace example {}
    • using声明机制 using std::cout; 仅引用需要的实体

  • 尽量使用const来替换define宏定义的使用

  • 内存泄漏–分配的内存未回收

  • malloc / free是C/C++语言的标准库函数;管理动态内存
    • 堆中空闲块组成的空闲链，malloc从链表头依次查找空闲块，符合申请空间的空闲块从链表取出进行分配； free将需要释放的块加入空闲链表示回收
  • new / delete是C++的运算符或表达式

  • 引用即是变量的别名，必须初始化，初始化后指向不会改变

  • 不要返回堆空间的引用，会引起内存泄漏

  • A，B..，X 逗号表达式，返回值为X,与A B .. 无关

  • C++ 通过函数名的改编实现函数重载

  • C/C++混合编程（）

  • 默认形参
    fun(int a1,int a2 … int ak…int an);
    如果ak设置了默认参数，则ak…an全部需要设置默认参数

  • 内联inline

    • 仅可使用在函数实现的前面 inline int f(){}；写在定义前无效。
    • inline 函数中不要写循环
    • 类内的函数默认inline函数

  • 内存空间（高->低）

    • 内核区:操作系统运行区
    • 栈：函数，变量
    • 堆：new/delete 的对象
    • 读写段（静态常量区）:全局，静态变量；生命周期到main执行结束
    • 只读段：字符串，程序代码
  • 拷贝
    • 浅拷贝 仅copy地址
    • 深拷贝 判断自复制==>销毁旧空间==>新开空间==>拷贝内容

  • 左值&右值

    • 无法取地址的值为右值 lvalue—常量、临时对象
    • 引用类型函数返回的不能是右值==> 需要返回静态变量
    • 普通函数的返回值是临时变量,不可取引用

  • 容器 vector (代替数组)

    • 动态扩容：size == capacity 时扩容（新开翻倍空间==>复制原来数据==> 回收原空间 ==> 在新空间添加新元素

  • 短路求值特性

    • if(0 && a++); //a++ 不计算
    • if(1 || a++); //a++ 不计算

  • 后置型表达式

    • a++ ==> 先返回a 在对a+1
    • 重载： Student opreator++(int){} — int 仅做区分不传参数
    • 返回临时变量不可取地址
  • 前置型(性能高)
    • ++a ==> 先+1 在返回(a+1)
    • 重载： Student& opreator++(){}
    • 效率高于后置
    • 可取地址
  • 函数调用
    • 通过对象调用函数–函数对象：携带状态(记忆性)
    • 直接通过函数名调用—全局(匿名)函数：无记忆性
  • 函数类型

类和对象

  • class 默认private ，struct 默认public

  • 构造函数初始化分配资源

    • 构造函数先执行初始化表达式,初始化的顺序与成员变量声明的顺序一致
    • 以下仅可以初始化列表初始化
      • const成员变量
      • 引用
      • 对象成员（括号内为复制，调用拷贝构造）

  • 拷贝构造函数

    • 对新对象进行赋值时调用；传递参数为对象时调用
    • 默认浅拷贝，因此涉及到内存空间操作时需要重写为深拷贝
    • 使用引用&符号为了防止无限递归调用
      • 注：仅在已有对象对新对象赋值才会调用，已经存在的对象之间赋值不调用

  • 析构函数

    • 回收分配给对象的资源不是对象本身；对象本身需要由free/delete来销毁
    • 栈，全局，静态 对象在程序结束是自动调用析构；堆对象 需要手动调用free/delete来销毁对象，销毁过程自动调用析构。

  • 赋值运算符函数

    • 已经存在的对象间赋值 [ a = b ] 操作
    • 可能导致内存泄漏，double free；必须显式定义重载(释放a空间；再深拷贝b)

  • 静态成员函数

    • 仅可以操作静态函数
    • 可以直接通过类名调用[Student::show()]，无需通过对象
    • 没有this指针
    • 虽然是静态的但是仍然放在代码段不是静态区
    • 仅可以在全局初始化,存储在全局静态区

注：区分
Student::show() //静态调用不会产生对象
Student().show() //匿名类调用，生成匿名对象，语句执行结束立即销毁

  • this指针形态

运算符重载

  • 友元（单向，不传递，不继承，不受public..的限制）
  • 内置类型运算符无法重载
  • . .* ?: :: sizeof不可重载
  • 不能有默认参数，操作数个数不会改变
  • 至少有1个枚举或自定义类型
  • 仅可以重载存在的运算符

  • 3 种重载方式

    • 普通函数(数据为public)
    • 成员函数

      可重载操作符：= ++ – += -> * () []
      类型转换函数必须用成员函数
      输出流操作数不能使用成员函数重载

    • 友元函数(数据为private)

      可重载操作符：== ！= < > <= >= << >>
      友元函数无this；成员函数有this

  • 类型转换函数

    • 成员函数
    • 没有参数，返回值
    • 函数内return 自身类型的对象
    • 尽量少用
  • new 分配堆空间==> 调用构造初始化 ==> 对象返回指针
  • delete 调析构回收成员变量空间 ==> 回收对象本身的空间

String

  1. 截取截取s中从pos开始的n个字符的子串
    s.substr(pos, n)
  2. 替换用s1替换s中从pos开始n个字符的子串
    s.replace(pos, n, s1)
  3. 查找查找s中第一次出现s1的位置，并返回
    s.find(s1)
  4. 返回一个指针，指向字符串 s1 中字符 ch 的第一次出现的位置
    strchr(s1, ch)
    返回一个指针，指向字符串 s1 中字符串 s2 的第一次出现的位置
    strstr(s1, s2)

  5. 长度 s.length() / s.size() /strlen(s)

  6. 复制字符串 s2 到字符串 s1
    strcpy(s1, s2)

  7. 连接字符串 s2 到字符串 s1 的末
    strcat(s1, s2)
  8. 连接
    s.append(str)
  9. 比较，如果 s1 和 s2 是相同的，则返回0；如果 s1< s2 则返回< 0；如果 s1>s2 则返回值> 0
    strcmp(s1, s2)
